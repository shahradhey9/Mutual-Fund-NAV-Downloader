# Mutual Fund NAV Downloader

A Python-based web application to search for Indian Mutual Funds and download their historical Net Asset Value (NAV) data.

## Features
- Search funds by name.
- View historical NAV data for specific date ranges.
- Download data as CSV.
- Modern, responsive UI.

## Tech Stack
- **Backend**: Python, Flask, mftool
- **Frontend**: HTML, CSS, JavaScript

## Installation
1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python app.py
   ```
3. Open browser at `http://localhost:5000`
